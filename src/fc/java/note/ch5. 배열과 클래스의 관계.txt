ch5. 배열과 클래스의 관계

Array와 class는 데이터를 담는 역할은 같지만 만들어지는 구조가 다름.
배열은 동일한 데이터를 담는 동일한 구조,
클래스는 서로 다른 데이터를 담는 이질적인 구조,

1.동일한 구조와 이질적인 구조
2.기본배열과 객체배열의 관걔
3.클래스를 이용한 논리적인 사고력

Quiz 1 . 동일한 기억공간을 메모리에 연속적인 구조로 만들어서 사용하는 자료구조는 무엇인가.
→ 배열
Quiz 2 . 현실세계의 객체는 이질적인 구조로 표현이 된다. 객체지향프로그래밍에서 이질적인 구조를 설계하는 도구는 무엇인가.
→ 클래스
Quiz 3. 배열과 클래스의 공통점으로 자바에서 배열과 클래스를 무엇으로 취급하는가.
→ 객체
Quiz 4 . 배열과 클래스로 객체를 생성할 때 공통적으로 사용하는 연산자는 무엇인가.
→ new
